Browse and import starter demo sites and page templates with Starter Blog with few clicks.

== Description ==
This plugin is an add-on for the Starter Blog WordPress Theme.

The demo content will make your website look like the preview of a theme so that you get a basic guideline for making your website.

Once installed and activated, browse and import demo sites from your WordPress backend with 4 steps:
1. Choice a site to import.
2. Install recommended plugins.
3. Import media and content.
4. Import config.

Done!

== Installation ==
1. Upload the entire `starter-templates` folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the `Plugins` menu in WordPress.

You will find `Appearance -> Starter Templates` menu in your WordPress admin panel.

== Upgrade Notice ==
You can update the plugin using "automatic WordPress updater".
